# Map<K,V> Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add real `Map<K,V>` behavior sufficient for compiler symbol tables and begin replacing `semantics.rs` sidecar logic.

**Architecture:** Add `MirType::Map`, lower `Map[K,V]` type expressions, and implement builtin-style `map_new`, `map_set`, `map_contains`, `map_get`, and `map_len`. Keep the first implementation minimal and behavior-proven; prefer a simple internal representation over a complete hash-table design.

**Tech Stack:** Rust-backed Axon compiler crates (`axon-mir`, `axon-codegen`), Axon source under `src/`, cargo tests from `rust-backed-compiler-for-axon/`.
