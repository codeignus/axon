# Sidecar Allowlist — Final State

This document lists every `*.rs` file allowed in `src/` after the zero-cargo cutover.
Any `.rs` file not on this list is a bug.

## Sidecars (all under `src/`)

| File | Responsibility | FFI exports |
|---|---|---|
| `clap.rs` | CLI argument parsing (clap) | via `#[axon_export]` |
| `tracing.rs` | tracing/log subscriber init | via `#[axon_export]` |
| `sidecar.rs` | panic handler, source formatting | `axon_fail`, `format_source_for_test`, `format_axon_file` |
| `compiler/syntax/lexer.rs` | file read + FFI to C lexer | `axon_lex_token_stream`, `axon_lex_all_tokens` |
| `compiler/syntax/parser.rs` | delimiter scan + parse entry | `run_parse_check`, `parse_stage_failed` |
| `compiler/proj/discover.rs` | directory list / canonicalize / exists | via `#[axon_pub_export]` |
| `compiler/proj/targets.rs` | target-resolution FS ops | `axon_classify_check_target_rs`, `axon_classify_test_target_rs` |
| `compiler/semantics/semantics.rs` | project FS walk + parse import bindings | `verify_project_calls`, `parse_import_bindings` |
| `compiler/semantics/ownership.rs` | snippet runner only | via `#[axon_pub_export]` |
| `compiler/diagnostics/diagnostics.rs` | color/tty detection, render line | `diag_error`, `stage_should_abort`, `axon_diag_note`, `axon_diag_warn`, `axon_string_starts_with`, `axon_string_concat` |
| `compiler/ir/ir.rs` | JSON encode helpers, write source file | `write_source_file`, `write_ir_cache`, `read_ir_cache`, `append_ir_module`, `ir_module_header`, `ir_local_entry`, `ir_assign_stmt`, `ir_return_terminator`, `ir_func_entry` |
| `compiler/backend/backend.rs` | process exec / chmod / atomic rename / preserve | `run_lowered_to_artifact`, `launch_self_built`, `preserve_suffixed_binary` |
| `compiler/backend/native_codegen.rs` | LLVM IR + object emit (inkwell) | `native_emit_object_for_module` |
| `compiler/backend/foreign_archive.rs` | cargo/go subprocess for user rust_deps/go_deps | `foreign_build_rust_bridge_archive` |
| `compiler/backend/toolchain.rs` | probe rustc/cc/go/llvm-config/clang | (internal helpers) |
| `compiler/pipeline_check/test_fmt.rs` | render diagnostic line (color) | `render_diagnostic_line`, `fmt_stage_summary` |
| `mcp/mcp.rs` | MCP server transport | `run_mcp_server` |
| `main.rs` | Axon entry point (generated) | `main` |

## Not on this list

These are **deleted** during the cutover:

- `src/Cargo.toml` — deleted in Phase B
- `src/Cargo.lock` — deleted in Phase B
- `src/axon_native_build_bin/main.rs` — deleted in Phase B (driver replaced by FFI)
- Any `.rs` file in `native-rust/` — deleted in Phase C

## Generated (gitignored, not tracked)

- `target/cache/app/rust/bridge/Cargo.toml` — generated by the Axon build system at build time
- `target/cache/app/rust/bridge/src/lib.rs` — generated bridge shim
